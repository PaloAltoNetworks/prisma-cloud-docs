=== Description 


Security definitions in the `security` section of a path or root should refer to an authentication scheme identified in the `securityDefinitions` section.

=== Fix - Buildtime

*OpenAPI* 


Example:
[source,yaml]
----

...
security:
- petstore_auth:
- write:pets
- read:pets

...
----
