== Supported Languages for Specific Domains on Prisma Cloud Code Security

Prisma Cloud Code Security supports a list of languages for IaC scans on specific domains.

* Terraform

* Terraform Plan

* CloudFormation

* AWS SAM

* Kubernetes

* Dockerfile

* Serverless framework

* Bicep

* ARM

* Argo Workflows

* Azure Pipelines

* BitBucket Pipelines

* Circle CI Pipelines

* GitHub Actions

* GitLab CI
